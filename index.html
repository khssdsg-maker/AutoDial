功能特性

多账号管理

支持添加、删除、切换多个宽带账号，一键拨号连接。

创建系统连接

直接在Windows中创建PPPoE拨号连接，无需手动设置。

开机自动拨号

开机登录后自动执行拨号，支持静默模式。

断线自动重连

检测到断线后自动重新拨号连接。

系统托盘

最小化到系统托盘，拨号成功/失败自动通知。

密码管理

密码显示/隐藏切换，导入/导出配置。

安装方法

方法一：PowerShell一键安装（推荐）

以管理员身份运行PowerShell，执行以下命令：

PowerShell
# 下载并安装 AutoDial
PS> irm https://raw.githubusercontent.com/khssdsg-maker/AutoDial/main/install.ps1 | iex

方法二：CMD一键安装

以管理员身份运行CMD，执行以下命令：

命令提示符
:: 下载并安装 AutoDial
C:\> curl -o %TEMP%\install.bat https://raw.githubusercontent.com/khssdsg-maker/AutoDial/main/install.bat && %TEMP%\install.bat

方法三：Winget安装（Windows 11）

Windows 11 用户可使用 Winget 包管理器：

终端
PS> winget install khssdsg-maker.AutoDial

方法四：手动下载安装

下载压缩包，解压后运行：

步骤
1. 下载 AutoDial.zip
2. 解压到任意目录
3. 双击 AutoDial.exe 运行
4. 勾选"开机自动拨号"即可

快速安装脚本

复制以下命令到PowerShell（管理员），一键完成下载、安装、创建快捷方式：

$dir = "$env:LOCALAPPDATA\AutoDial"; New-Item -ItemType Directory -Force -Path $dir | Out-Null; Invoke-WebRequest -Uri "https://github.com/khssdsg-maker/AutoDial/releases/download/v2.5/AutoDial.exe" -OutFile "$dir\AutoDial.exe"; Invoke-WebRequest -Uri "https://github.com/khssdsg-maker/ADial/releases/download/v2.5/AutoDial.ico" -OutFile "$dir\AutoDial.ico"; $ws = New-Object -ComObject WScript.Shell; $s = $ws.CreateShortcut("$env:USERPROFILE\Desktop\宽带拨号.lnk"); $s.TargetPath = "$dir\AutoDial.exe"; $s.IconLocation = "$dir\AutoDial.ico,0"; $s.Save(); Write-Host "安装完成！桌面已创建快捷方式。"

常见问题

Q: 软件无法运行？

A: 确保已安装 .NET Framework 4.0（Windows 10/11 默认自带）

Q: 拨号失败？

A: 检查连接名称、用户名、密码是否正确，或查看错误提示

Q: 如何取消开机自启动？

A: 在软件中取消勾选「开机自动拨号」→ 保存设置

Q: 如何备份配置？

A: 点击「导出」保存账号文件，在新电脑点击「导入」恢复
