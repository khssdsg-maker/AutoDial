# AutoDial - 宽带自动拨号工具

一个简洁的 Windows 宽带自动拨号工具，支持多账号管理、开机自动拨号、静默拨号、断线自动重连等功能。

## 功能特性

- **多账号管理** - 支持添加、删除、切换多个宽带账号
- **一键拨号** - 选择账号后一键连接宽带
- **开机自动拨号** - 开机登录后自动执行拨号
- **静默拨号** - 后台自动拨号，不弹出界面窗口
- **断线自动重连** - 检测到断线后自动重新拨号连接
- **连接状态监控** - 实时显示连接状态、在线时长、网络速度
- **系统托盘** - 最小化到系统托盘，不占任务栏
- **错误码解释** - 显示中文错误说明，而非错误代码
- **导入/导出配置** - 支持备份和迁移账号设置
- **鼠标悬停提示** - 所有按钮和选项都有功能说明

## 使用方法

### 直接使用
1. 下载 `AutoDial.exe`
2. 双击运行
3. 点击「新增账号」
4. 输入连接名称、用户名、密码
5. 点击「保存设置」
6. 点击「拨号连接」

### 设置开机自动拨号
1. 打开软件
2. 勾选「开机自动拨号」
3. 点击「保存设置」

### 设置静默拨号
1. 先勾选「开机自动拨号」
2. 再勾选「静默拨号」
3. 点击「保存设置」

### 设置断线自动重连
1. 勾选「断线自动重连」
2. 点击「保存设置」
3. 断线后软件会自动尝试重新连接

### 最小化到托盘
1. 点击窗口右上角的关闭按钮
2. 软件会最小化到系统托盘（右下角）
3. 双击托盘图标可重新打开窗口
4. 右键托盘图标可快速操作

### 导入/导出配置
1. 点击「导出配置」保存当前账号到文件
2. 在新电脑上点击「导入配置」选择文件
3. 账号会自动导入（不会覆盖已有账号）

## 系统要求

- Windows 10 / 11
- .NET Framework 4.0（Windows 默认自带）

## 编译方法

如果需要自行编译，使用 Windows 自带的 C# 编译器：

```batch
C:\WINDOWS\Microsoft.NET\Framework64\v4.0.30319\csc.exe /target:winexe /out:AutoDial.exe AutoDial.cs /r:System.Windows.Forms.dll /r:System.Drawing.dll
```

## 文件说明

- `AutoDial.exe` - 主程序
- `AutoDial.cs` - 源代码
- `%APPDATA%\AutoDial\accounts.txt` - 账号配置文件
- `%APPDATA%\AutoDial\settings.txt` - 设置文件
